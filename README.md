## Заполнение .env
Заполните логин и пароль от своей базы данных в файле .env или вставте свою ссылку

---

## Запуск API 

Что бы запустить скрипт в dev режиме в команндную строку напишите

> #### npm run dev

Чтобы запустить скрипт в продакшн используйте команду

> #### npm start
